# 手工回归 — 动态 / 通知

### MC-DYN-01：动态首屏加载
- 步骤：登录后进入 DynamicPage。
- 期望：1s 内显示事件流；存在缓存时先展示缓存。

### MC-DYN-02：下拉刷新 / 上拉加载更多
- 步骤：在 DynamicPage 反复下拉、上拉。
- 期望：刷新出新条目；上拉到底部追加分页；不出现重复。

### MC-DYN-03：事件跳转
- 步骤：依次点击 PR / Issue / Push / Star 类型条目。
- 期望：跳转到对应详情页正确。

### MC-DYN-04：通知中心
- 步骤：进入 NotifyPage → 标记单条已读 / 全部已读。
- 期望：UI 即时变化；返回后再次进入仍保持已读状态。
