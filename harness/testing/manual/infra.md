# 手工回归 — 基础设施

### MC-INFRA-01：弱网降级
- 步骤：开飞行模式 → 启动 App。
- 期望：所有缓存可用页面显示缓存；UI 不崩溃；恢复网络后能自动刷新。

### MC-INFRA-02：i18n 即时切换
- 步骤：切换语言。
- 期望：任意已打开页签的标题、按钮文案立即更新。

### MC-INFRA-03：Realm 兼容
- 步骤：覆盖安装新版本（保留旧 Realm 文件）。
- 期望：不崩溃；如有 schema 变更走迁移逻辑。

### MC-INFRA-04：深色 / 高对比模式（可选）
- 步骤：在系统中切换深色。
- 期望：当前不要求黑暗模式适配，但不应出现白底白字等不可读情况。
